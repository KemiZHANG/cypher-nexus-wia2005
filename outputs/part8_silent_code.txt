Part 8 - The Silent Code

Mission Problem
Scan intercepted messages for trigger phrases and rank messages by threat severity.

Algorithm Used
Brute Force String Matching + Threat Ranking

Rejected Algorithms
- KMP is rejected for this small demonstration because optimization is less important than transparent ranking logic.
- Hash set lookup is rejected because it cannot naturally scan full text for multi-word trigger phrases.

Dataset
dataset file: Part 8.xlsx
dataset source: D:\Datasets (2).zip

row count: 8
columns used: Message_ID, Text_Stream, Route_Tag

Key Result
Highest-ranked message: M04 with threat score 7.

Result Explanation
Each message is scanned with manual brute force phrase matching, then ranked by phrase weights and route-tag evidence.

Time and Space Complexity
Time O(messages*phrases*text_length*phrase_length), space O(messages+phrases).
